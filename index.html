CSS Snippet Cheatsheet

Ever have trouble recalling the exact syntax for your favourite CSS code? Give it a permanent home and add it to this page! Select any snippet below and it'll automatically select all of the code for you to copy.

Flex Grid Gradients Shadows

Flex

flexbox row

Use these three properties to create a Flexbox row layout. 

.row {
    display: flex;
    flex-direction: row;
    flex-wrap: wrap;
}

flexbox column

Use this to create a Flexbox column layout.

.column {
    display: flex;
    flex-direction: column;
}

Center content

Use this to perfectly center elements on the page. 

.center {
    display: flex;
    justify-content: center;
    align-items: center;
}

Grid

CSS grid layout

Build a 12-column layout using CSS Grid.

.grid {
    display: grid;
    width: 100%;
    grid-template-columns:
    repeat(12, 1fr);
                  }

Define Grid

Defines the rows and columns of the grid.

.grid {
    display: grid;
    grid-template-columns: 
     12px 12px 12px;
    grid-template-rows: 
     12px 12px 12px;
}

Centered Grid

Builds a CSS Grid in which elements are centered. 

.grid {
    display: grid;
    grid-template-columns:
    repeat(12, 1fr);
    justify-items: center;
    align-items: center;
}

Linear Gradients

Linear gradient

This will create a background linear gradient from top to bottom. 

.linear-gradient-background {
  background-image: 
    linear-gradient(
    rgba(232, 102, 236, 0.3) 0%,
    rgba(232, 102, 236, 0.6) 100%
    );
}

Radial Gradient

This will create a radial gradient.

.radial-gradient {
    background: radial-gradient(
        #e66465, 
        #9198e5
        );
}

conic gradient

This will create a conic gradient.

.conic-gradient {
    background: conic-gradient(
        red, orange, yellow, 
        green, 
        blue
        );
}

Shadows

Box Shadow

This will create a simple box-shadow.

.box-shadow {
    box-shadow: 10px 5px 5px red;
}

Inset Box shadow

This will create a box-shadow on the inside. 

.inset-box-shadow {
    box-shadow: inset 5em 1em gold;
}

Drop Shadow

This will create a drop-shadow, following the shape of an image rather than the box itself. 

.drop-shadow {
    filter: drop-shadow(
        30px 10px 4px #4444dd
        );
}
